Esercizio 1

Si implementi la seguente funzione C:

int matrix_min(int** dest, int** a, int** b, int rows, int cols);

Ove a e b sono due matrici in input aventi dimensione rows * cols.

La funzione deve creare in dest un array contenente gli elementi di a che sono minori o uguali ai corrsipondenti valori in b. In altre parole, se a[r][c] <= b[r][c] allora dest dovrà contenere il valore a[r][c]. La funzione deve ritornare la dimensione dell' array dest.

Le matrici devono essere scorse per riga.

Esempio

a =
[3 1]
[6 0]

b =
[2 5]
[4 0]

dest = [1 0]

La funzione restituisce 2.

Esercizio 2

Le matrici sparse sono particolari matrici in cui molti degli elementi sono uguali a zero. In questo esercizio gli elementi diversi da zero di una matrice sparsa sono rappresentati con la struttura SparseMatrixElement contenente gli indici di riga e colonna (row e col) dell' elemento, il suo valore value, ed il campo next che punta al prossimo elemento della matrice diverso da zero. Note le dimensioni di una matrice sparsa, la si può rappresentare come una lista di strutture SparseMatrixElement:

struct SparseMatrixElement {

  int row;                   

  int col;                   

  float value;               

  SparseMatrixElement* next; 

};

typedef struct SparseMatrixElement SparseMatrixElement;

typedef SparseMatrixElement* SparseMatrix;

Si implementi la seguente funzione C:

float** sparse_to_dense_matrix(SparseMatrix sparse_mat, int rows, int cols);

ove sparse_mat è una matrice sparsa in input (lista di strutture SparseMatrixElement) di dimensioni rows * cols.

La funzione deve allocare e restituire in output una nuova matrice mat, di dimensione rows * cols, in cui gli unici elementi diversi da zero corrispondano in valore a quelli contenuti in sparse_mat.

Esempio

sparse_mat = [row: 0, col: 0, value: 1.0] --> [row: 0, col: 2, value: 2.0] --> [row: 1, col: 2, value: 5.0] --> [row: 2, col: 2, value: 3.0] --> NULL

rows = 3

cols = 3

mat =

[1.0 0.0 2.0]

[0.0 0.0 5.0]

[0.0 0.0 3.0]

Esercizio 3

Si consideri la definizione di matrice sparsa dell' esercizio precedente, implementare la seguente funzione ricorsiva C:

int sparse_matrix_contains_element(SparseMatrix sparse_mat, float value);

che data una matrice sparsa in input sparse_mat (lista di strutture SparseMatrixElement) e un valore di ricerca value, restituisca 1 se sparse_mat contiene un elemento di valore value, 0 altrimenti.

Esempio

parse_mat = [row: 0, col: 0, value: 1.0] --> [row: 0, col: 2, value: 2.5] --> [row: 1, col: 2, value: 5.0] --> [row: 2, col: 2, value: 3.0] --> NULL

value = 2.5

La funzione restituisce 1.
